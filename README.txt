En el siguiente README general se va a explicar el funcionamiento del juego SoulMirror.

- A continuación vamos a explicar los objetos que podemos encontrar en el juego:

-Cubos morados: Son NPC con los que podemos interactuar si pulsamos la tecla Espacio.

-Cubos rojos: Son enemigos que si te acercas de hacen daño

-Cubos naranja: Son elementos que pudes coger con el Espacio y lanzar con el click izquierdo.

-Cubo marrón: Objeto arrastable que puede accionar mecanismos, para poder arrastrar tenemos que estar cerca del cubo y pulsar Espacio

-Cubo azul cyan: Objeto botón que cuando es presionado produce una acción. 

-Esfera amarilla: Es la llave con la que podremos abrir las puertas amarillas

-Esfera verde: Es la poción que se recogera automaticamente nada más tocarla.

-Pared amarilla: Puerta que impide el paso y solo se puede abrir con la llave

-Pared azul: Puerta que impide el paso y que solo se puede abrir accionando un botón.

-Puerta verde: Cambio a una zona distinta

-Puerta negra: Significa el final del prototipo.

Las teclas 1 2 3 4 5 significan respectivamente espada,escudo,arco,bomba,pocion. 
Se puede ir intercambiando entre ellas pulsando el correpsondiente número.
Si por ejemplo tenemos seleccionado el 1 y pulsamos 2 para poner el escudo tendremos
que ultilizar el click izquierdo para realizar la acción. 

Tecla Shift: Se corre a más velocidad.
Tecla Enter: Sirve para cerrar ventanas emergentes.

Para ejecutar el prototipo: Dentro de la carpeta SoulMirror ejecutar ./soulMirror

Las asignaturas de TDS,TAG,V1 y V2 son las que estan relacionadas con el código.
